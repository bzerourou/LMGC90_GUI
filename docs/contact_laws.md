# Lois de Contact

Définition des interactions entre vos avatars.

## Lois disponibles
Seulement trois types d'inetractions qui sont implémenter : 
- IQS_CLB (frottement Coulomb)
- IQS_CLB_g0 (frottement Coulomb avec un distance de dépard)
- COUPLED_DOF (liaison parfaite)

## Paramètres
- Nom : chaine de cinq caractères
- Fric (pour IQS_CLB ou IQS_CLB_g0 )

## Utilisation typique
- COUPLED_DOF pour joints mécaniques
- IQS_CLB pour contacts frottants
