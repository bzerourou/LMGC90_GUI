# Post-traitement

Définition des sorties pour LMGC90.

## Commandes disponibles
- SOLVER INFORMATIONS
- TORQUE EVOLUTION (sur avatar/groupe)
- BODY TRACKING (suivi de corps)
- KINETIC ENERGY, etc.

## Fonctionnalités
- Fréquence d’écriture (step)
- Sélection d’avatar ou groupe pour certaines commandes

## Génération
Bouton **Générer Datbox** utilise ces commandes
